# Jotai - атомарное управление состоянием для React

## Описание

Jotai - это библиотека для управления состоянием в React, основанная на атомарном подходе. Каждый атом представляет собой независимую единицу состояния.

## Когда использовать

- ✅ Когда нужна гибкость в управлении состоянием
- ✅ Для композиции состояния из атомов
- ✅ Когда нужна производительность

## Базовое использование

### Установка

```bash
npm install jotai
```

### Создание атомов

```javascript
import { atom, useAtom } from 'jotai';

const countAtom = atom(0);

function Counter() {
  const [count, setCount] = useAtom(countAtom);

  return (
    <div>
      <div>Count: {count}</div>
      <button onClick={() => setCount(c => c + 1)}>+</button>
      <button onClick={() => setCount(c => c - 1)}>-</button>
    </div>
  );
}
```

## Преимущества

- ✅ Атомарность - независимые единицы состояния
- ✅ Композиция - можно комбинировать атомы
- ✅ Производительность - обновляются только затронутые компоненты
